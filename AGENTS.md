# 融景 — Codex 项目说明

## 项目概述

PPT 场景合成工具：将 PPT 截图或视频录屏通过透视变换嵌入实拍背景图的屏幕区域，批量生成合成图片或视频。

## 重要约束

- 图像处理已去除 cv2，统一使用 PIL（Pillow）
- 视频合成的 PTS 必须用帧计数器，不能复制输入 PTS
- 模板路径在系统 Application Support 目录（Mac: `~/Library/Application Support/融景/templates/`）
- CLI 入口：`python cli.py`（图片合成 / 视频合成 / 模板管理）
